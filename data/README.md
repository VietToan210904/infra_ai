# InfraAI Data

Shared project data lives here when it is not specific to one app.

- `mock/` stores synthetic demo data.
- `seeds/` stores seed fixtures for future services.
- `geojson/` stores reusable geospatial fixtures.
- `samples/` stores small sample payloads and outputs.

The current frontend still keeps its app-specific synthetic TypeScript data in
`apps/web/src/data`.
