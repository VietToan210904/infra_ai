# Scripts

Reserved for repeatable project scripts.

- `dev/` for local developer helpers
- `ci/` for CI helpers
- `data/` for data-processing wrappers
- `copy_osm_geojson_to_web.py` copies processed OpenStreetMap GeoJSON into the Vite public directory.
- `copy_external_geojson_to_web.py` copies processed external-source GeoJSON into the Vite public directory.
- `validate_verification_geojson.py` validates authority/provider verification GeoJSON metadata.
- `copy_verification_geojson_to_web.py` copies validated verification GeoJSON into the Vite public directory.
